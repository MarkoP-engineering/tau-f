This is a repository for the development of a model rocket for the Czech Rocket Challenge 2026 competition.

The rocket will have to ignite it`s engine, launch, climb up to 800 meters, release the parachute and return safely.

The fuselage will be 3D printed out of Prusament ASA due to it's low density and extreme durability. The whole body will weight about 650g
The bodz will be split into multiple sections - below they are in order form bottom to top
- Tail + fin module - a part that contains the stabilizing aerodynamic elements of the rocket and will help keep the rocket engine in place
- Engine + egg section - a large section the engine is screwed into from the bottom that also contains sace to put a chicken egg to be flown on the rocket
- Barometer section - a section for the mandate barometer to be mounted
- Avionics section - will contain most of the elctronics needed for the rocket to function - custom flight controller, camera, battery and a connector for external power while on standby
- Nosecone and ejection system - a system toggled electrically and powered by a spring for the release of the parachute to assure the safe return of the rocket





We would like to thank Prusa Research for providing us with the filament necessary to make this project possible!
